# Troubleshooting

Use the focused guides in this directory when `agenthub doctor`, compose, or runtime startup points you to a `docs/troubleshooting/...` link.

- `environment-setup.md` — missing Node.js, Python, or `opencode`
- `compose-failures.md` — missing profiles, bundles, or broken compose prerequisites
- `plugin-degraded-mode.md` — runtime config or local plugin bridge problems
- `guard-and-skill-conflicts.md` — missing guards, mixed OMO/native profiles, or inherited OMO baseline issues
- `model-configuration.md` — invalid `provider/model` syntax in overrides

For OMO mixed-profile warnings, use `guard-and-skill-conflicts.md`.
