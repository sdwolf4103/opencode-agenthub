# opencode-agenthub v0.2.1

Released: 2026-04-04

## Highlights

- Makes the built-in `plan` and `systems-architect` specialists able to save planning artifacts under `docs/superpowers/plans/` without granting broad repo write access.
- Fixes stale built-in `demo-coding-team` showcase seeds so existing HR homes refresh to the current managed showcase version automatically.

## Detailed changes

### Path-scoped planning writes

- Updated the built-in demo coding showcase so `plan` and `systems-architect` use OpenCode native path-scoped `edit` permissions.
- Limited those two specialists to `docs/superpowers/plans/*` and `docs/superpowers/plans/**` while keeping `write: "deny"` and `bash: "deny"` in the generated bundle permissions.
- Aligned supporting souls, checklist text, and promotion memo wording with the new write model.

### Built-in showcase reseeding

- Updated HR showcase seeding so `demo-coding-team` is refreshed when the stored showcase seed marker does not match the current package version.
- Added regression coverage for the stale-seed case, including the exact old marker shape that existed before this release.
