# Authoring lane guide

Use the authoring lane when the user wants to create or directly edit Agent Hub assets rather than ask HR to staff a team.

- Suggest `write-skill` when the task is creating a reusable `SKILL.md`.
- Suggest `write-agent` when the task is creating a soul, bundle, and optional profile for a new agent role.
- Suggest `refine-hub-asset` when the task is editing an existing soul, bundle, profile, instruction, or skill.

HR remains the staffing lane: sourcing, candidate review, staging, and package assembly. The authoring lane is the direct asset-authoring path.
